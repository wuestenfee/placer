# placer

Placer is a basic layout manager inspired by pythons tkinter grid layout manager.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     placer:
       github: wuestenfee/placer
   ```

2. Run `shards install`

## Usage

```crystal
require "placer"
```

Checkout examples/label_frame.cr
